# react-github-profile-app-revamped

Revamped Vite + React demo app (plain CSS) with mock localStorage authentication and protected routes.

Features
- Register / Login (mocked using localStorage)
- Protected route for repositories search (only accessible after login)
- Profile view & edit
- Repository Search (GitHub Search API) with full details (stars, forks, language, description)

Quick start
1. npm install
2. npm run dev
3. Open http://localhost:5173
